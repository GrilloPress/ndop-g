Design hypotheses

November

  1. If we add a page list to our local header, then we will improve navigation because users will easily be able to find further information about how the NHS handles their data
  2. If we add a back link to the top of our content pages, then we will improve navigation because users can quickly jump back
  3. If we add breadcrumbs to the top of each content page, then we will improve navigation because users will be able to just back to the landing page in one click
