Design hypotheses

November

In lab 6, users found navigating between pages easier but were unsure which page contained the information they wished to see.

If we update the page headers to reflect the subjects in the users' own voice
then we will improve navigation
because users will easily be able to find further information that is relevant to them

In lab 6, users didn't use the bottom of the page pagination links. Some users on UserZoom used them extensively.

If we increase the size of the next/prev label and reduce the page title slightly
then users will better understand how to cycle through pages
because users will immediately understand the concept that the bottom of the page link takes them to the next page/topic.

Previous (unproven)

If we add breadcrumbs to the top of each content page, then we will improve navigation because users will be able to just back to the landing page in one click
